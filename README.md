# Hotel Reservation Management System

A full-stack hotel booking system built with Laravel, covering both backend logic and frontend UI.

## 🚀 Overview
This system allows users to search, book, and manage hotel reservations while providing administrators with tools to manage rooms, bookings, and user interactions.

## ✨ Features
- 🔐 User registration & authentication
- 🏨 Room listing and availability checking
- 📅 Booking system with check-in/check-out logic
- 💳 Booking confirmation & payment status
- 🧑‍💼 Admin dashboard for booking and room management
- 📧 Email confirmation after booking
- ⭐ Review submission system

## 🧠 Key Highlights
- Full system design including ERD and use case diagrams
- Structured development process (proposal → implementation → report)
- Clean MVC architecture using Laravel
- Responsive UI built directly with Tailwind CSS

## 🛠 Tech Stack
- Laravel (PHP)
- MySQL
- Blade Template Engine
- Tailwind CSS
- JavaScript (minimal interactivity)
- XAMPP (local development)
